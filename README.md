🟦 Identität des Moduls
4all ist das Universal‑Modul der iki1uc‑Struktur.
Es trägt alle Module, alle Serien, alle Respons‑Rahmen, alle NC‑Slots, ohne selbst eine Richtung zu setzen.

Der Name bedeutet:

4 = vier Grundebenen (Frame, Flow, Switch, Merge)

all = alle Module, alle Wege, alle Übergänge

4all ist das Modul, das alles tragen kann, weil es nichts erzwingt.

Darum ist die Leere korrekt, neutral, MARKT‑3‑fähig.
